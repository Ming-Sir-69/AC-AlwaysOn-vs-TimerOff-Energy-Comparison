5. **综合报告** — PPT + Excel + 交互仪表盘 + 学术版

## 核心结论

- **外出时长是唯一显著因子**（Beta=0.848，解释84.8%方差）
- **28°C 是热惯性激活阈值** — 低于此温度建筑自身保温已足够
- **高温长时场景(>34°C, >8h)关空调可节能 45-55%**
- **短暂外出(<30分钟)不应关机** — 重启能耗大于维持能耗
